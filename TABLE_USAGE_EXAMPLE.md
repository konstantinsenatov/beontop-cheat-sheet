# Использование таблиц в BeOnTop HTML Generator

## Простое использование

Теперь вы можете просто:

1. **Добавить таблицу через меню "Вставка → Таблица"** в Google Docs
2. **Заполнить таблицу данными**
3. **Таблица автоматически отобразится в превью!**

## Автоматические функции

- ✅ **Автоматическое определение заголовка**: если первая строка содержит короткие слова (< 15 символов), она считается заголовком
- ✅ **Автоматическое создание секции**: таблицы вне секций автоматически становятся отдельными секциями
- ✅ **Адаптивный дизайн**: таблицы корректно отображаются на мобильных устройствах

## Расширенное использование

Если нужны дополнительные настройки, используйте секции:

```
/* start */
LAYOUT: Table
TITLE: Моя таблица
TABLE STYLE: bordered
TABLE HEADER: on
ALIGN: center

[Создайте таблицу в Google Docs]

CTA: Подробнее | /link
/* end */
```

## Доступные стили таблиц

- `default` - базовый стиль
- `bordered` - с границами
- `striped` - зебра (чередующиеся строки)
- `hover` - эффект при наведении
- `compact` - компактная таблица
- `spacious` - просторная таблица

## Директивы

- `TABLE STYLE: default|bordered|striped|hover|compact|spacious`
- `TABLE HEADER: on|off` (по умолчанию автоматическое определение)
- `ALIGN: left|center|right`
